## 核心理念
### 1. 没有缺点，只有用错的特质。
“制约”也可以被解读为“资源”，优势运用不当，也可能带来负面后果。
任何看似在拖后腿的倾向，都问自己一个问题：“它在什么情境下曾帮到我？”



## 案例
例：希望能改善学习状态
```
目标：希望能改善学习状态
个性化学习方案：告诉自己“我是在完成明天的任务”，而不是今天的任务。
特质：安全感来源——资源富足量
理论：用户的安全感来自于提前量，例如，时间、金钱充足时能保持平静、专注，反之则陷入惴惴不安。可能起源于高中经历，工作日提前做周末的作业，有助于周末好好玩。
起到的效果有：
（1）解读/叙事重构
	完成任务的感受就从“被迫为现在吃苦”变成了“主动替未来减负”。
（2）带来一种放松和掌控的感觉，因为今天的任务已经完成了，我在做的是明天的任务。而不是被任务悬在心上的感觉压着。
	这个对我确实能起到激励效果，可能源自于高中经历中，今天做明天的作业，工作日提前做周末的作业，有助于明天更轻松。另外一点是，“没有缺陷，只有用错的特质”，“活在未来”也是可以利用的。
```
再比如：想要更努力
```
目标：希望能让自己更努力
个性化激励方案：用“我不想xx，所以要努力”而不是“我想要xx，所以要努力”来激励自己。
特质：逃避驱动
```

## 行动步骤
### 1. 明确目标
### 2. 识别关键特质
### 3. 落地为具体的行动方案
**特质驱动的行为设计**
尊重个人的特质，具体问题具体分析，而非通用方案，例如，利用“逃避驱动”特质——用“我不要”而不是“我要”驱动。此时应给人一种“认识自己”或者“啊哈时刻”的感觉。
自下而上、从个体经验中归纳，而非自上而下地套用。
**优势本位干预**
一句话：不用盯着缺点修修补补，而是用好你已经有的东西。
**为特质匹配一个心理杠杆**
包括但不限于叙事重构 、动机框架转换等
